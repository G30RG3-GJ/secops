## ⚙️ Installation / ინსტალაცია

### Method 1: Auto-Update (Recommended)
1. Install any previous version
2. Open app → Settings → Check for Updates
3. The app will automatically download and install the latest version

### Method 2: Direct Download
1. Download `SecOps-v4.0-debug.apk` from [Releases](https://github.com/G30RG3-GJ/secops/releases)
2. Enable "Install from Unknown Sources" in Android Settings
3. Install the APK

### Method 3: Build from Source
```bash
git clone https://github.com/G30RG3-GJ/secops.git
cd secops
# Create local.properties with your SDK path:
echo "sdk.dir=C:\\Users\\YourUser\\AppData\\Local\\Android\\Sdk" > local.properties
.\gradlew assembleDebug
```
